An NVMe-oF Discovery Controller is a repository of information necessary for Hosts to connect to NVMe-oF subsystems.

The DEM Discovery Controller is a centralized implementation of an NVMe-oF Discovery Controller.

This tool:

  • Lists which Hosts and which Targets are managed by this DEM

  • Notifies Hosts of resource changes on provisioned NVMe-oF subsystems

  • Visualizes groups which consist of Endpoints and the Hosts provisioned with them
