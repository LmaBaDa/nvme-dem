An NVMe-oF Discovery Manager is centralized repository of information necessary for Hosts to connect to NVMe-oF subsystems.

The Distributed Endpoint Manager (DEM) is an implementation of the NVMe-oF Discovery Manager.

This tool:

  • Lists which Hosts and which Targets are managed by this DEM

  • Notifies Hosts of resource changes on provisioned NVMe-oF subsystems

  • Visualizes groups which consist of Endpoints and the Hosts provisioned with them
